# Project-Portfolio
Collection of group and individual university projects, plus other personal projects.\
The **Thesis** folder contains the full pdf and the powerpoint presentation I used for the discussion.
If interested, the source code can be viewed at the [following GitHub repository](https://github.com/Andreaduva/CSPH).\
Title: “Simulating Fluid Dynamics Effects on Small Bodies using N-Body codes” 
- Study on the validity of a fluid-dynamics approach to simulating cometary atmospheres. 
- Development of an add-on module based on the Project Chrono multiphysics library to numerically solve the fluid equations using the Smoothed Particle Hydrodynamics (SPH) method.

The **University projects** folder lists some of the group and individual works I have done. More details are given inside that directory.
